💡 Quick Documentation & Workflow Explanation

1 Define the Rulebook

Define variables for styles, characters, and environments inside the left column. Shorthand tag labels (e.g., `[CHAR-SPY]`) are auto-assigned to these entries.

2 High-Speed Keyboard Tags

Type a bracket symbol [ inside any shot description box. A dynamic suggestion list will immediately pop up! Select with your keyboard arrows and hit Enter.

🗣️ Spoken Dialogue Rule

To add spoken audio, select a character tag and append the speech snippet to write `[CHAR-TAG] says, "spoken line"`. The compiler will automatically extract this into a dedicated audio channel.

MD What is an "MD" File?

MD refers to Markdown. It is a highly optimized structured format ideal for feeding visual contexts into LLMs like Claude, Gemini, or ChatGPT with zero formatting loss.
